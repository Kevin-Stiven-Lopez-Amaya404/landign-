# Imagenes

Imagenes optimizadas usadas por el sitio.

## Organizacion

- `brand/`: logo e identidad visual.
- `hero/`: imagen principal de la landing.
- `content/`: imagenes de secciones, actividades y momentos.
- `team/`: fotos de integrantes del equipo.

## Buenas practicas

- Usar nombres descriptivos.
- Mantener imagenes optimizadas.
- Agregar `alt` descriptivo cuando la imagen se use en HTML.
- Evitar archivos originales pesados si no se van a usar en produccion.
